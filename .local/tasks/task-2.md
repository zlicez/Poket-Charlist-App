# Редизайн UI/UX — чистый современный дизайн

  ## What & Why
  Полный аудит выявил критические проблемы в типографике, цветовой системе, визуальной иерархии и адаптивности. Нужен комплексный редизайн с применением лучших практик: правильная типографическая шкала, семантические цвета через тему, чёткая визуальная иерархия, современный чистый стиль.

  ## Done looks like
  - Шрифты заменены на современные: Inter/DM Sans для UI, Lora для заголовков, JetBrains Mono для чисел — все загружаются оптимально (только нужные)
  - Минимальный размер текста 12px, чёткая типографическая шкала (12/14/16/20/24/32)
  - Все хардкодные цвета (amber-*, green-*, red-*) заменены на семантические переменные темы
  - HP-бар, модификаторы, состояния используют CSS-переменные, корректно переключаются в тёмной теме
  - Визуальная иерархия: главные блоки (характеристики, HP, КД) визуально отделены от второстепенных (заметки, союзники, фракции)
  - Карточки характеристик компактнее на мобильных — 2 колонки на среднем экране, свёрнутые навыки по умолчанию
  - Единый ритм отступов по шкале 4/8/12/16/24/32px
  - Навигация по секциям (sticky tabs или прокрутка к якорям) на длинной странице
  - Кнопка режима редактирования понятна: иконка + текст "Играть"/"Редактировать" даже на мобилке
  - Меньше визуального шума: бейджи объединены, убрана избыточная информация
  - Пустое состояние нового персонажа с подсказками

  ## Out of scope
  - Функциональные изменения (новые вычисления, API, бэкенд)
  - Новые страницы или маршруты
  - Переделка drag-and-drop
  - Изменение структуры данных

  ## Tasks
  1. **Оптимизация шрифтов** — Убрать 22 неиспользуемых шрифта из загрузки, заменить font stack на Inter (body), Lora (headings), JetBrains Mono (numbers/dice), обновить CSS-переменные --font-sans, --font-serif, --font-mono.

  2. **Типографическая шкала** — Заменить все text-[8px], text-[9px], text-[10px] на минимум text-xs (12px). Внедрить системную шкалу размеров: xs(12)/sm(14)/base(16)/lg(20)/xl(24)/2xl(32). Убрать произвольные пиксельные размеры.

  3. **Семантические цвета** — Ввести CSS-переменные для состояний: --color-positive, --color-negative, --color-hp-bar, --color-hp-fill, --color-ability-bg, --color-ability-border и т.д. Заменить все ~44 хардкодных bg-amber-*, text-green-*, text-red-* на семантические классы. Обеспечить корректную работу тёмной темы.

  4. **Визуальная иерархия карточек** — Разделить карточки на 3 уровня: primary (характеристики, бой, HP — акцентная рамка/тень), secondary (оружие, снаряжение, навыки — стандартная карточка), tertiary (заметки, союзники, фракции — минимальный стиль). Добавить заголовки секций.

  5. **Улучшение мобильной вёрстки** — Навыки свёрнуты по умолчанию на мобилке. Единый ритм gap/padding по шкале 4px. Кнопка режима с полным текстом. Группировка и подписи секций для навигации. Sticky-навигация или прокрутка к секциям.

  6. **Очистка компонентов** — Уменьшить количество бейджей в оружии/снаряжении (объединить в одну строку). Добавить пустое состояние для нового персонажа с подсказками. Улучшить карточки персонажей на главном экране.
